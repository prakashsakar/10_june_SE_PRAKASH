Ans:Object-oriented programming-As the name suggest uses object in programming.Object-
oriented programming aims to implement real-word entities like inheritance,hiding,
polymorphism,etc in programming.The main aim of oop is to bind together the data and 
the functions that operate on them so that no part of the code can access this data
except that function.
